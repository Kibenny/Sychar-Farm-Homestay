```
│   └── main.css     ← Shared styles

└── README.md
```

## Deploying to Netlify (Free)
1. Go to https://netlify.com and sign up (free)
2. Click "Add new site" → "Deploy manually"
3. Drag the entire `sychar-farm` folder into the upload area
4. Your site is live instantly at a `.netlify.app` URL

## Adding a Custom Domain
1. Buy domain at https://namecheap.com (e.g. sycharfarm.com ~KES 1,500/yr)
2. In Netlify: Site settings → Domain management → Add custom domain
3. Follow the DNS instructions — live within 24hrs

## Updating the Site
- Edit the HTML files on your computer
- Re-drag the folder to Netlify to redeploy (takes seconds)
- OR connect to GitHub for automatic deploys on save

## Images
All photos are embedded directly in the HTML as base64 — no separate image files needed.
This means the site works fully offline and deploys as a simple folder with no dependencies.
